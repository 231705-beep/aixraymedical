# AI Medical X-Ray System - Frontend

This is a modular, enterprise-grade React frontend for an AI-powered Radiology Diagnosis System.

## Technologies Used
- **React 19** with **Vite**
- **Plain CSS Modules** (Modular styling)
- **Framer Motion** (Smooth animations)
- **Lucide React** (Medical icons)
- **React Router Dom** (Centralized routing)

## How to Run the Project

1. **Open your terminal** in the root directory of the project.
2. **Navigate to the frontend folder**:
   ```bash
   cd frontend
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Start the Development Server**:
   ```bash
   npm run dev
   ```
5. **Access the App**:
   Open [http://localhost:5173](http://localhost:5173) in your browser.

## Key Features
- **Public Landing Page**: Overview of the AI system and "Get Started" flow.
- **Patient Portal**: X-ray upload, AI + Doctor report tracking, specialist recommendations, and appointment booking.
- **Doctor Portal**: Clinical workspace for reviewing X-rays, comparing AI results, and signing prescriptions.
- **Admin Portal**: System control, user management, and doctor verification.
